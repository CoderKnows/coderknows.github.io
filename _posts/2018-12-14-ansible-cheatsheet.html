Install Ansible

sudo apt-get update
sudo apt-get install software-properties-common
sudo apt-add-repository --yes --update ppa:ansible/ansible
подключение репозитория
sudo apt-get install ansible
установка Ansible

Ansible Galaxy

Устанавливает в каталог /home/<user-home-dir>/.ansible/roles/ playbook-и из сервиса galaxy.ansible.com

ansible-galaxy install nginxinc.nginx
Официальный playbook от NGINX.

Ansible Inventory

/etc/ansible/hosts
файл с перечнем хостов и серверов
localhost ansible_connection=local
хост для локального подключения (через UNIX-socket)
ansible-inventory --list
выводит список хостов в json-формате
