mpm

npm list -g --depth=0
Выводит список npm-пакетов.
-g - пакеты, установленные глобально.
--depth - глубина отображения зависимостей. Без этого параметра отобразить все пакеты со всеми зависимостями.

Установка Node.js

sudo npm cache clean -f
sudo npm install -g n
sudo n stable
Установка стабильной версии node.js через npm
sudo npm cache clean -f
sudo npm install -g n
sudo n 4.4.2
Установка LTS-версии node.js через npm. В примере установилась версия 4.4.2
